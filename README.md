# Automatic Street Light System (IoT)

## Overview
This project is an IoT-based Automatic Street Light System developed using NodeMCU ESP8266, an LDR sensor, and LEDs. The system automatically turns the street lights ON during darkness and OFF during daylight, helping to save energy.

## Features
- Automatically turns ON lights in darkness.
- Automatically turns OFF lights during daylight.
- Energy-efficient street lighting.
- Uses an LDR sensor to detect light intensity.
- Low-cost and easy-to-implement IoT solution.

## Components Used
- NodeMCU ESP8266
- LDR Sensor
- LEDs
- Resistors
- Breadboard
- Jumper Wires

## Software Used
- Arduino IDE

## Working
- The LDR sensor detects the surrounding light intensity.
- During daylight, the system keeps the LEDs OFF.
- When it becomes dark, the NodeMCU automatically turns the LEDs ON.

## Applications
- Smart Cities
- Smart Street Lighting
- Parking Areas
- Garden Lighting
- Energy Saving Systems
